#### Overview
Our project analyzes the quality of wines from the UCI Wine Quality dataset and allows the data to be manipulated by the user.
The UCI dataset contains chemical measurements for major components of red and white wine variations, including acidity, sugar content, alcohol content, pH, and sulfur content, which can be used to determine the quality of the wine. 

#### Structure

##### Project Setup
Imports the UCI dataset and ensures it is available for reference
##### Load and Clean Dataset
Loads the .csv files from the imported dataset and cleans the data for use.
##### Analyze the Cleaned Dataset
Summarizes the cleaned data through statistical analysis, comparing the overall statistics of the red and white wines while locating the Top 5 quality wines from each variation.
##### Visualize the Results
Bar charts displaying the distribution of quality scores for red and white wines in the dataset.
##### Quality Prediction and Wine Recommendation
Introduces an input-based feature. Predicts wine quality based on features input by the user and recommends similar wines, recommends a wine based on single-feature preferences, and can randomly choose a wine from the dataset for the user.
